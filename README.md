# ASAntiCrash
一个反崩溃的工具, 通过导入文件自动执行
可以防止包括: NSObject, 
            NSArray, 
            NSMutableArray, 
            NSSet, 
            NSMutableSet,
            NSDictionary, 
            NSMutableDictionary, 
            NSString, 
            NSMutableString, 
            NSAttributedString, 
            NSMutableAttributedString, 
            NSURL, 
            NSFileManager, 
            NSCache
            等的基本崩溃发生
